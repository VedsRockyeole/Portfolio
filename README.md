# Vedant Yeole — Portfolio

A single-page developer portfolio built with plain HTML, CSS, and JavaScript — no frameworks, no build step. Themed around a "blueprint / build-log" aesthetic: a technical grid backdrop, an animated terminal intro, and each section framed like a system module (README, Stack, Build Log, Shipped, Education).

**Live site:** https://vedsrockyeole.github.io/portfolio/

---

## Sections

| Module | Content |
|---|---|
| `01 · README` | About me — background, focus areas, quick stats |
| `02 · STACK` | Technical skills — languages, core CS, web, databases, tools, working style |
| `03 · BUILD LOG` | Work experience — internships with measurable outcomes |
| `04 · SHIPPED` | Projects — E-Commerce Website, Deep Learning-Based Pothole Detection |
| `05 · EDUCATION` | Academics |
| `CONTACT` | Email, phone, LinkedIn, GitHub |

## Tech

- **HTML5 / CSS3** — layout, custom properties, responsive grid
- **Vanilla JavaScript** — terminal typing effect, scroll-reveal via `IntersectionObserver`
- **Fonts** — Space Grotesk (display), Inter (body), JetBrains Mono (data/labels)

No dependencies, no build tools — open `index.html` directly in any browser.

## Run locally

```bash
git clone https://github.com/VedsRockyeole/portfolio.git
cd portfolio
open index.html   # or just double-click the file
```

## Deploy

This repo is deployed with **GitHub Pages**:

1. `Settings → Pages`
2. Source: **Deploy from a branch**
3. Branch: `main`, folder: `/ (root)`

Any push to `main` updates the live site automatically.
